一 部署环境
   我们了解到Hexo是基于Node.js的它是一个基于ChromeV8引擎
的JavaScript运行环境，可在非浏览器环境下，解释运行JS代码。
所以在搭建博客之前需要先去官网下载node.js，下载LTS(长期支持版)，
安装完成之后会有两个组件 分别是node.js本身和npm 包管理器，
命令行使用 node -v ，npm -v来查看node的版本和包管理器版本
其次我们还需要安装git和注册一个github
二 初始化博客
  1. 我们是借助于npm包管理器来安装Hexo博客框架的，由于国内镜像源速度太慢，
    所以在安装之前先利用 npm来安装 cnpm 
    代码： npm install -g cnpm --registry=https://tegistry.npm.taobao.org
    使用cnpm -v查看版本号
    2.使用cnom来全局安装HEXO博客框架
    cnpm install -g hexo-cli
    使用hexo -v 查看版本号
    3.创建一个文件夹
    mkdir bolg 然后 cd进入blog pwd查看路径
    4.使用hexo生成博客
    使用 sudo hexo init 在blog中生成hexo博客
    (sudo：在 macOS 或者 Linux使用 windows不需要)
    安装完成后会提示：INFO State blogging with Hexo！
    使用 ls -l 查看hexo自动生成的文件
    使用 hexo s 启动博客
三 写博客
1.使用 hexo n "博客名称" 来新建博客
 在blog文件夹的source中的_posts 找到新建的博客进行编辑
四 把个人博客部署到远端
把博客部署到github上{
   1.在github上创建一个新的仓库
   2.在blog文件夹中安装git部署插件
        cnpm install --save hexo-deployer-git
    3.设置blog中的_config.yml
      把_config.yml文件滑到最低部进行配置
      # Deployment
        ## Docs: https://hexo.io/docs/deployment.html
          deploy:
          type: git
          repo: https://github.com/likai1123/likai1123.github.io.git // 仓库地址
          branch: master


}
